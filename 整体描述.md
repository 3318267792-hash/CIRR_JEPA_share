# CIRR Retrieval 项目整体描述

## 1. V0：Baseline 闭环搭建

整体结构为：

* `target image -> frozen CLIP-L image encoder -> target_emb`
* `reference image + modification text -> frozen CLIP -> 简单融合 / MLP -> pred_emb`
* 用 `pred_emb` 去对齐 `target_emb`

### 主要实现

* `datasets/cirr_dataset.py`
* `models/clip_target_encoder.py`
* `models/baseline_query_encoder.py`
* `train_cirr_baseline.py`
* `eval_cirr_baseline.py`

### 结果

| Version     | Recall@1 | Recall@5 | Recall@10 | Recall@50 | Recall Subset@1 | Recall Subset@2 | Recall Subset@3 |
| ----------- | -------- | -------- | --------- | --------- | --------------- | --------------- | --------------- |
| V0 Baseline | 0.117675 | 0.372160 | 0.527625  | 0.821335  | 0.353265        | 0.602009        | 0.774456        |

---

## 2. V1：在 Baseline 上加入 temperature / logit scale

整体结构为：

* `target image -> frozen CLIP-L image encoder -> target_emb`
* `reference image + modification text -> frozen CLIP -> 融合模块 -> pred_emb`
* 训练时使用：`scale * (pred_emb @ target_emb.T)`

### 主要实现

* 保留原有 `train / eval` 主体逻辑
* 在 query encoder 中加入 `logit_scale`

### 结果

| Version        | Epochs | Total Queries | Recall@1 | Recall@5 | Recall@10 | Recall@50 | Recall Subset@1 | Recall Subset@2 | Recall Subset@3 |
| -------------- | ------ | ------------- | -------- | -------- | --------- | --------- | --------------- | --------------- | --------------- |
| V1 Logit Scale | 8      | 4181          | 0.195408 | 0.500359 | 0.656063  | 0.904568  | 0.480029        | 0.702224        | 0.835924        |
| V1 Logit Scale | 2      | 4181          | 0.1490   | 0.4286   | 0.5831    | 0.8756    | 0.4171          | 0.6518          | 0.8063          |

---

## 3. V2：Attention 融合版本

整体结构为：

* `target image -> frozen CLIP-L image encoder -> target_emb`
* `reference image + modification text -> frozen CLIP -> attention / transformer 融合 -> pred_emb`
* 用 `pred_emb` 去对齐 `target_emb`

### 主要实现

* 将 `ref_emb` 和 `text_emb` 作为两个 token
* 使用小型 attention / transformer block 做融合
* 训练与评测流程整体沿用 baseline 框架

### 结果

| Version             | Epochs | Total Queries | Recall@1 | Recall@5 | Recall@10 | Recall@50 | Recall Subset@1 | Recall Subset@2 | Recall Subset@3 |
| ------------------- | ------ | ------------- | -------- | -------- | --------- | --------- | --------------- | --------------- | --------------- |
| V2 Attention Fusion | 2      | 4181          | 0.054054 | 0.210954 | 0.343459  | 0.681655  | 0.220282        | 0.419039        | 0.615642        |

---

## 5. V4：Qwen-family 双边 Embedding Frozen Baseline

整体结构为：

* `reference image + modification text -> Qwen3-VL-Embedding-2B -> z_q`
* `target image / gallery image -> Qwen3-VL-Embedding-2B -> z_t`
* 检索时使用：`sim = z_q @ z_t.T`

### 结果
